# Specification

## Summary
**Goal:** Update navigation bar with title text in mobile view and extend bottom CTA bar to desktop view.

**Planned changes:**
- Add white "Mitsubishi Srikandi Subang" title text to the Navigation Bar in mobile view with appropriate font sizing
- Remove "Simulasi Kredit" menu item from the Navigation Bar

**User-visible outcome:** Users will see the dealership name in the mobile navigation bar, a cleaner menu without the credit simulation option, and desktop users will have quick access to the call-to-action buttons at the bottom of their screen.
